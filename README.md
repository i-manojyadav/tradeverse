# Tradeverse

A modern full-stack cryptocurrency trading platform built using the MERN Stack.

## Features

- User Authentication
- Live Cryptocurrency Prices
- Buy & Sell Crypto
- Portfolio Management
- Transaction History
- Responsive UI

## Tech Stack

### Frontend
- React
- Vite
- JavaScript
- CSS

### Backend
- Node.js
- Express.js
- MongoDB
- Mongoose

## Future Improvements

- Real-time WebSocket updates
- Advanced trading charts
- Price alerts
- Two-factor authentication
